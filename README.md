# smget
Download videos from your favorite social networks.

## installation

1- clone the repository
```bash
git clone https://github.com/voidh7/smget && cd smget
```

2- install dependencies
```bash
chmod +x smget.sh && ./smget.sh install
```

# use of the tool
download media video
```bash
./smget.sh video <link>
```

download media in audio
```bash
./smget.sh audio <link>
```
